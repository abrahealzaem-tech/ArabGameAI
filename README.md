# ArabGame AI
محرك لصنع الألعاب باستخدام الذكاء الاصطناعي، مع إمكانية رفع الصور، التحكم في اللاعب، اختيار جودة اللعبة، ومعاينة المستقبل الجميل.

## المميزات
- حركة اللاعب + الاصطدامات
- دعم AI بالعربية والإنجليزية
- رفع الصور وإضافتها كعناصر
- اختيار جودة اللعبة: 360p / 720p / 1080p / 4K
- معاينة المستقبل الجميل
